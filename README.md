# Global Superstore SQL Analysis

This project performs an end-to-end SQL analysis on the Global Superstore dataset to extract business insights across sales, customers, products, and operations.

## Key Analyses
- Sales performance by year, quarter, region, and category  
- Top products and customers by revenue and profit  
- Customer retention and repeat vs one-time customer analysis  
- Product profitability and discount effectiveness  
- Shipping cost, delivery efficiency, and cost-effective ship modes  
- Seasonal trends and moving average analysis for sales smoothing  

## Tools & Skills
- SQL (Postgres)
- CTEs, Window Functions, Aggregations
- Business-focused analytical thinking
- sqlalchemy
